## *Built for ET Gen AI Hackathon*

An AI-powered financial assistant that helps users understand their money habits, identify financial gaps, and take actionable steps toward a better financial future.

---

## 🌐 Live Demo

👉 [Try the Website](https://ai-money-mentor-nj45.vercel.app/)

---

## 🧠 The Problem

In India, a majority of individuals lack a structured financial plan — not due to lack of income, but due to lack of awareness and guidance.

* Financial advice is expensive 💰
* Tools are complex and intimidating 📊
* Beginners don’t know where to start ❌

---

## 💡 Our Solution

**AI Money Mentor** simplifies personal finance using AI.

It transforms raw financial inputs into:

* Clear insights
* Personalized advice
* Practical action plans

All within a simple, user-friendly interface.

---

## ✨ Key Features

### 📊 Financial Health Score

A dynamic score (out of 100) based on:

* Savings behavior
* Expense patterns
* Financial discipline

---

### ⚠️ Smart Problem Detection

Identifies exactly where users go wrong:

* Lack of emergency fund
* Overspending habits
* Poor financial planning

---

### ✅ Actionable Improvements

Simple, beginner-friendly steps:

* No jargon
* No complexity
* Just practical advice

---

### 🤖 AI-Powered Advisor

Powered by Google Gemini API:

* Personalized financial guidance
* Context-aware responses
* Interactive chatbot for real-time queries

---

### 🔮 Future Simulation

Visualizes two paths:

* Current behavior → Financial stress
* Improved habits → Financial stability

---

### 📅 30-Day Financial Plan

Structured weekly roadmap to improve financial health step-by-step.

---

### 📩 Email Report

Users receive a complete financial summary via EmailJS:

* Score
* Problems
* Improvements
* AI-generated advice

---

## 🏗️ System Architecture

The application follows a **frontend-driven architecture with AI integration**:

* User inputs are processed using JavaScript logic
* Financial insights are generated instantly
* AI enhances personalization using Gemini API
* EmailJS handles report delivery

---

## 🔄 Workflow

1. User completes financial questionnaire
2. System calculates financial health score
3. Problem areas are identified
4. Improvement suggestions are generated
5. AI provides personalized financial advice
6. Results are displayed interactively
7. User receives a detailed report via email

---

## ⚙️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **AI Integration:** Google Gemini API
* **Email Service:** EmailJS
* **Architecture:** Frontend-driven, API-integrated

---

## ⚠️ Error Handling

* API failure → fallback advice shown
* Email failure → retry alert
* Invalid input → validation checks
* Network issues → graceful handling

---
